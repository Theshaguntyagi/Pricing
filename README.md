# Pricing Card Project

A simple and stylish pricing card web application that showcases different subscription plans for a fitness service.

## Introduction

This project provides a clean and visually appealing pricing card display for showcasing subscription plans. It's built with HTML, CSS, and a touch of JavaScript to enhance the user experience.

## Features

- Responsive design for various screen sizes.
- Three subscription plans: Basic, Pro, and Ultimate.
- Stylish color schemes and hover effects.
- Information about each plan, including features and pricing.
- CTA buttons for easy navigation.

## Access

You can check the website by [clicking here](https://theshaguntyagi.github.io/Pricing).

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/theshaguntyagi/Pricing.git

2. Open the project folder:
   cd pricing-card

3. Open the index.html file in your web browser.

## Usage

Simply open the index.html file in your web browser to view the pricing card. You can customize the content, styles, and links according to your needs.

## Customization

Feel free to customize the project to suit your preferences. Here are some areas you might want to modify:

1. *Content:* Update the subscription plans, prices, features, and links in the HTML file.
2. *Styles:* Adjust the colors, fonts, and other styles in the style.css file.
3. *Author Credits:* Change the author credits in the HTML file to give appropriate credit.
